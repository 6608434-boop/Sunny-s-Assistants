# Sunny's Assistants ☀️

**AI-ассистенты для разработчиков и бизнеса**

Солнце в центре. Планеты вокруг. Бесконечные возможности.

---

## Философия

- **Солнце** = клиент в центре
- **Планеты** = сервисы и гейты вокруг
- **Соты** = контейнеры для гейтов
- **Гейты не знают друг о друге** — только через соты
- **Связи между сотами** — визуальные (орбиты, кривые)
